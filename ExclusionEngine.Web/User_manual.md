# Exclusion Engine User Manual

This guide covers daily use of the Exclusion Engine portal.

## Dashboard quick actions
- **Customer Data** opens the full customer grid.
- **User Manual** opens this document.
- **Log out** ends your session.

## Customer Entry
1. Select a **Client**.
2. Enter customer details.
3. Click **Validate + Save**.
4. If prompted by CASS, choose standardized or original address.

### Phone field behavior
- The phone field auto-formats as `XXX-YYY-YYYY`.
- Extensions are supported, e.g. `5551234567 ext 42` becomes `555-123-4567 ext 42`.

## Recent Entries
- Use **Search** / **Clear** to filter.
- **Edit** loads a row back into Customer Entry.
- **Delete** removes the row after confirmation.

## Customer Data page
- Use filters, sorting, and paging to find records.
- Use row checkboxes and **Delete Selected** for deletions.
- **Edit** opens the selected entry on the dashboard for update.

## Password reset
- On login, choose **Forgot password?**
- Request a reset link and complete reset on the reset page.

## Admin areas
Admins can manage users and clients from **User Admin** and **Client Admin**.
